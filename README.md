## Como rodar

**Pré-requisitos:** Node.js 18+

```bash
# 1. Instale as dependências
npm install

# 2. Inicie o servidor de desenvolvimento
npm run dev
```

Acesse **http://localhost:5173** no navegador.

## Build para produção

```bash
npm run build
```

Os arquivos gerados ficam na pasta `dist/`, prontos para deploy estático (GitHub Pages, Vercel, etc.).
